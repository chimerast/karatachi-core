このサイトは

たけうち(chimerast)が自由気ままに作っているWicket用ライブラリのサンプルページです。 Wicketを使うとこんなこともできるよというサンプルになればと思っています。 ライブラリおよびサンプルにはソースコードも含まれているため、自由に中身を見ることもできますが、 右上にソース表示リンクがついているので、利用してみてください。

ライブラリおよびサイトのライセンスについて

org.karatachi以下のパッケージについては、TAKEUCHI Hideyuki (chimerast)が著作権を保持し、 CPLライセンスの元に配布します。 ライブラリに含まれている、prototype.js、script.aculo.us、syntaxhilighterについては、 それぞれのライセンスが適用されます。

ライブラリ取得使用方法

一応下記URLをたどればjarファイルのダウンロードを行うことができますが、 依存しているjarファイルが多いのでMaven2を使用してライブラリの取得を 行うことを推奨します。Maven2の使用方法は他のサイトを参照してください。

Maven2用設定

レポジトリ

Mavenレポジトリとして下記を追加することで、karatachi.orgが管理する ライブラリをMavenを通して使用可能な状態になります。

karatachi-wicket

Wicket用の便利なコンポーネント集です。最新バージョンは0.3-SNAPSHOTになっていますが、 SNAPSHOTは気が向いたときにアップデートされ、クラス名などもがんがん変わるため、 気になる場合は0.2.25を使用してください。

依存でkartachi-*がいくつか入ります。
